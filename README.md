# QRCM-QQReplyClassMsg
这是一个能恢复班级群消息+弹出群消息的回复助手，用于在电教上课时可以及时回复消息并且看到消息。|A reply assistant that restores and displays class group notifications, perfect for quickly checking and replying to messages during tech-assisted lessons.
这个自动回复助手会在老师发送消息之后弹出一个小窗口以便查看老师发来的消息。既让老师感到安心，也不会打扰到正在上课的老师，也帮助了电教快速回复消息。
本程序使用了NapCat的部分接口(不然怎么实现发送和接收消息)
程序可以自定义NapCat的websocket端口，回复消息的内容，以及设置qq群号
